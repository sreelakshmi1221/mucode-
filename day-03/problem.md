## 📌 Problem: Move All Zeros to End

### 🧾 Problem Statement
Given an array of integers, move all the zeros to the end of the array while maintaining the relative order of the non-zero elements.

You must perform this operation **in-place** without using extra space for another array.

---

### 📥 Input Format
- First line contains an integer `N` — size of the array  
- Second line contains `N` space-separated integers  

---

### 📤 Output Format
- Print the modified array after moving all zeros to the end  

---

### ⚙️ Constraints
- `1 ≤ N ≤ 10^5`  
- `-10^9 ≤ A[i] ≤ 10^9`

---

### 🧪 Examples


## Input:

5
0 1 0 3 12

---
## Output:

1 3 12 0 0

---

---

### 📝 Instructions
- Try solving before the mentor session

### 🚀 Challenge
Can you solve this in **O(N)** time with **O(1)** extra space?
